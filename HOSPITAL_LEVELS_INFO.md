# ข้อมูลระดับโรงพยาบาล (Hospital Levels)

เอกสารนี้อธิบายคำจำกัดความของระดับโรงพยาบาลที่ใช้ในระบบ One Phitsanulok

## 1. SAP Level (Service Affiliate Plan)
ใช้สำหรับการจัดสรรทรัพยากรและงบประมาณ (แสดงด้วยเฉดสี Indigo ในระบบ)

| Level | รายละเอียด |
|-------|------------|
| **P+** | ศูนย์ความเชี่ยวชาญระดับสูง (Super Tertiary) |
| **P** | ศูนย์ความเชี่ยวชาญ (Tertiary) |
| **A+** | โรงพยาบาลทั่วไปขนาดใหญ่ขีดความสามารถสูง |
| **A** | โรงพยาบาลทั่วไป (General Hospital) |
| **S+** | โรงพยาบาลชุมชนแม่ข่าย (Node) |
| **S** | โรงพยาบาลชุมชน (Community Hospital) |

## 2. Size Level
แบ่งตามขนาดเตียงและขีดความสามารถ (แสดงด้วยเฉดสี Purple ในระบบ)

| Level | ขนาด | รายละเอียด |
|-------|------|------------|
| **A** | > 500 เตียง | โรงพยาบาลศูนย์/โรงพยาบาลทั่วไปขนาดใหญ่ |
| **S** | 120-500 เตียง | โรงพยาบาลทั่วไปขนาดเล็ก |
| **M1** | 90-120 เตียง | โรงพยาบาลชุมชนขนาดใหญ่ |
| **M2** | 60-90 เตียง | โรงพยาบาลชุมชนระดับกลาง |
| **F1** | 30-60 เตียง | โรงพยาบาลชุมชนขนาดเล็ก |
| **F2** | 10-30 เตียง | โรงพยาบาลชุมชนขนาดเล็กมาก |
| **F3** | < 10 เตียง | โรงพยาบาลส่งเสริมสุขภาพตำบล/ศูนย์สุขภาพชุมชน |
| **P** | - | ภาคเอกชน (Private) |

## การแสดงผลใน Application
- **Hospital Page**: แสดงการจัดกลุ่มโรงพยาบาลตาม SAP และ Size Level
- **One LR/OR/ICU/IPD**: แสดงข้อมูลทรัพยากรรายโรงพยาบาล
- **Waiting & Visit**: แสดงสถิติการให้บริการ
